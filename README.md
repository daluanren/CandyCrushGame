## 简介 ##

### 开发环境 ： ###
　　Win 7 + VS2013 + Unity 4.6.3

###游戏简介： ###
　　消除类游戏，游戏玩法参考CandyCrush，当横排或者竖排出现相同糖果的时候会被消除，得分。

* 游戏开始界面
* 游戏界面
* 游戏配置界面（后期制作配套）

1. 生成新的Candy，并创建Candy类；
2. 按行列随机生成多个Candy；
3. Candy间的互换；
4. ...（后期继续补充）
